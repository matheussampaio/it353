Question 4:

Node.js is a JavaScript runtime build on Chrome's V8 JavaScript engine. Node.js uses an event-driven, non-blocking I/O model that makes it lightweight and efficient. NPM (Node Package Manager) is the largest ecosystem of open source libraries in the world.

You can use Node for developing backend, command line tools, desktop softwares (with NW.js or Electron), JavaScript libraries, web frameworks (Express) and many more. That's why Node is getting more and more popular.

Last november, Node.js merged with IO.js (the most popular Node.js fork) and published the version 4, adding support for ECMAScript 2015 (ES2015) among other things.
